# 기본 타입

## 문자열 선언방식

```ts
const str: string = "hello";
const num: number = 10;
```

## 배열 선언방식

```ts
const arr: Array<number> = [1, 2, 3]; // 숫자형 배열 방식 1

const items: numbdr[] = [1, 2, 3]; // 숫자형 배열 방식 2

const devs = (Array<string> = ["jay", "amie"]);
```

## 튜플 (다중 타입)

```ts
const tupleArr: [string, number] = ["hello", 123];
```

## 진위값 (boolean)

```ts
const show: boolean = true;
```

## 객체

// 1
const obj: object = {};
